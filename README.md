# Collage-Site

A simple and elegant HTML-based collage site for displaying images and creative content in an organized, visually appealing layout.

## 📋 Description

Collage-Site is a lightweight web project built with HTML that provides a clean interface for showcasing image collages. Whether you're displaying a portfolio, photo gallery, or creative works, this project offers a straightforward structure to get started.

## 🚀 Features

- **HTML-based**: Pure HTML structure for simplicity and easy customization
- **Responsive design**: Works across different screen sizes
- **Easy to customize**: Simple code structure makes it easy to modify and adapt
- **Lightweight**: Fast loading with minimal dependencies

## 🛠️ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- A text editor (VS Code, Sublime Text, etc.) for editing

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Pooja-79/Collage-Site.git
   cd Collage-Site
   ```

2. Open the HTML file in your browser:
   - Simply double-click the main `.html` file, or
   - Right-click and select "Open with" → your preferred browser

## 💻 Usage

1. Edit the HTML files to add your images and content
2. Customize the styling with CSS
3. Update image paths to point to your own image files
4. Open the file in a browser to preview your collage

## 🎨 Customization

You can customize:
- Image paths and sizes
- Layout and spacing
- Colors and fonts
- Add additional HTML elements as needed
